# Santa-Kebaya
Di Santa Kebaya, setiap jahitan adalah karya seni. Kami percaya bahwa kebaya bukan sekadar pakaian, tetapi simbol keanggunan dan kepercayaan diri seorang wanita. Dengan desain eksklusif dan detail yang dikerjakan penuh ketelitian, kami menghadirkan kebaya yang memperindah setiap momen berharga Anda.
